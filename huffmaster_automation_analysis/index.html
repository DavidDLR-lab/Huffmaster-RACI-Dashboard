How each task was scored

This is a first-pass model, not a commitment. Scores come from the action’s verb + noun, its stage/swimlane context, and the systems named or implied — cross-referenced to the July 2026 integrations diagram. Every task shows its three sub-scores and a rationale so an SME can adjust. Treat the task list as a working draft to review.
Claude Cowork — semi-automated

Claude drafts the document, does the research, or prepares the analysis; a person reviews and decides. No build, no integration — usable immediately. Signals: create, generate, draft, write, determine, identify, assess, review, compile, summarize, recap.

AI Agent (SWORD) — fully automated

A triggered agent runs the task end-to-end without a person — update a system, post a job, notify, track, validate against a rule. Requires the SWORD / Cascade Agents orchestration layer. Signals: update, notify, send, post, track, monitor, assign, schedule, validate, close, deactivate + recurrence cues (daily, per shift, per candidate).

Better Integration — eliminates the task

The data already lives in two systems and is being moved by hand; a connector removes the step entirely. Signals: names a system or is a handoff between two (ATS, HCM, CRM, PM Tool, Corpay, job boards); retype, copy to, load to, sync, close out. The diagram’s Essium → UKG Pro handoff is the flagged top target.

Human-led — stays manual

Physical deployment work (transport, on-site, staging), client relationships and negotiations, and point-of-decision judgment. Not an automation target.

Impact, effort & priority

Impact
Volume × criticality. Recurring / per-candidate / daily work scores High; per-engagement setup lower; on-critical-path items get a bump.
Effort
Cowork = Low (no build). AI Agent = Medium (needs SWORD). Integration = Medium, or High where the system is legacy/closed (Sabre, faxing).
Priority
Quick win = high impact + low effort · Strategic build = high impact + real build · Planned build = medium/medium · Easy fill-in = low effort · Deprioritize = low impact + high effort.
